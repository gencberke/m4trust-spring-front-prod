# Generated Core API types

`core-api.d.ts` is generated from `../../../contracts/openapi/core-api-v1.yaml`
by `npm run generate:api`. Generated files in this directory are never edited
manually. Public API models must be imported from the generated output rather
than recreated as parallel handwritten TypeScript types.

The Actuator readiness response is operational and intentionally remains a
small local type under `features/readiness`; it is not part of the public
OpenAPI contract.
